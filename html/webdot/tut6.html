WebDot can convert a graph into a "Tclet" for the Tcl Plugin. For example, tut6.dot contains: 

    digraph G {
        node [URL="\N.html"]
        edge [URL="\N.html"]
        {a b c} -> d
    }
and the source of this page contains (for the Tclet which should be displayed below): 
    <webdot src=/webdot/tut6.dot engine=neato type=tcl target=new>

after serving this page through /cgi-bin/webdot, this <webdot...> tag is replaced by: 

    <embed width=600 height=400 src="/cgi-bin/webdot/http://127.0.0.1/webdot/tut6.dot.neato.tcl">
