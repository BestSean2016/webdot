WebDot - Tutorial 1 [next]
Given a description of a graph in the dot language, e.g.: 
    graph G {
        a -- b -- c -- d
        b -- e -- d
    }

in a file that can be served by your web server, e.g. /webdot/tut1.dot

Then this image reference in an html web page: 

    <img src=/cgi-bin/webdot/webdot/tut1.dot.neato.png>

will produce the image at the right. The image reference consists of four parts: 

    /cgi-bin/webdot       The WebDot CGI server.
    /webdot/tut1.dot      The graph source file.
    .neato                The layout engine (either ".dot" or ".neato")
    .png                  The output format.
