[previous] [next] WebDot Tutorial: Embedded SVG graph
WebDot can render graphs in SVG or SVGZ formats for use in <object> or <embed> tags. For example this graph: 
digraph G {
	node [URL="\N.html"]
	edge [URL="\N.html"]
	a -> b -> c
	a -> c
}

can be generated with: 
<object
    data="/cgi-bin/webdot/webdot/svgembed.dot.dot.svgz"
    type="image/svg+xml"
    width="250" height="250">
<embed src="/cgi-bin/webdot/webdot/svgembed.dot.dot.svgz"
    type="image/svg+xml"
    coding="gzip"
    palette="foreground">
</embed>
</object>

Note the html trick of nesting <embed> inside <object> to get embedded graphs to work in non-HTML-4.01 browsers.

Note that edges are easily clickable in this format.
