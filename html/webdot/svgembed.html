WebDot also supports SVG or SVGZ as output formats. For example this graph: 
digraph G {
	node [URL="\N.html"]
	edge [URL="\N.html"]
	a -> b -> c
	a -> c
}

can be generated for web browsers that support SVGZ viewing with: 
<embed src=/cgi-bin/webdot/webdot/svgembed.dot.dot.svgz 
      type="image/svg-xml"
      coding="gzip"
      palette="foreground"
>

Note that edges are easily clickable in this format.
