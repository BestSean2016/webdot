WebDot also supports SVG as an output format. For example this graph: 
digraph G {
	node [URL="\N.html"]
	edge [URL="\N.html"]
	a -> b -> c
	a -> c
}

can be generated for web browsers that support SVG viewing with: 
<embed src=/cgi-bin/webdot/webdot/svgembed.dot.dot.svg 
      type="image/svg-xml"
      palette="foreground"
>

Note that edges are easily clickable in this format.
